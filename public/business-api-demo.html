业务 API Key 图片元数据示例

输入远程服务地址、业务 API Key 和仓库名称后，页面会调用开放 API 获取图片元数据，并在下方直接显示图片墙。你可以切换使用原始链接或 CDN 链接作为展示地址。

等待输入参数后开始请求。
提示：`repo_name` 为服务端记录中的仓库名，不是完整的 `owner/repo`。如果你的业务 API Key 只允许某几个仓库， 页面只能读取这些仓库里的数据。
请不要直接双击以 `file://` 方式打开这个页面。请通过本地图床前端服务访问，例如 http://localhost:5173/business-api-demo.html，否则浏览器会把本地文件当成独立安全域处理，导致请求或跳转异常。
